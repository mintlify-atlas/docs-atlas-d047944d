---
title: "Quickstart: subtitles playing in under five minutes"
description: "Create subtitle data, wire up LocalAudioSubtitles in a LocalScript, and start LocalAudio on the server to get animated subtitles running immediately."
---

This guide walks you through a complete working setup from scratch. By the end, audio tracks in your game will display animated subtitles through Subtitle Engine whenever LocalAudio fires a subtitle event.

<Tip>
  Make sure Subtitle Engine is installed in `ReplicatedStorage` and has a **"Self"** profile with no speaker name configured before starting. See the [Installation](/installation) page if you have not done this yet.
</Tip>

<Steps>
  <Step title="Create your subtitle data module">
    Create a `ModuleScript` named `Subtitles` inside `ReplicatedStorage > Data`. This module defines your speakers, macros, levels, and per-track subtitle entries.

    The following example sets up two speakers with colors, a reusable macro for an announcement chime, and two subtitle levels:

    ```luau Subtitles.luau
    return {
        Speakers = {
            Keepsake = {
                Color = Color3.fromRGB(0, 170, 255),
                Modifiers = {
                    Chorus = {
                        Color = Color3.fromRGB(30, 200, 255),
                    },
                    Background = {
                        Color = Color3.fromRGB(0, 140, 225),
                    },
                },
            },
            ASAS = {
                Color = Color3.fromRGB(200, 200, 200),
            },
        },
        Macros = {
            NormalAnnouncementChime = {
                Time = 0,
                Duration = 1,
                Message = "(Announcement chime)",
                MessageColor = Color3.fromRGB(0, 170, 255),
                Level = "ExtraAnnouncement",
            },
            WarningAnnouncement = {
                MessageColor = Color3.fromRGB(255, 170, 0),
                Level = "Dialog",
            },
        },
        Levels = {
            Dialog = 1,
            ExtraAnnouncement = 2,
        },
    }
    ```

    <Note>
      The `Levels` table maps level names to numeric values. A higher number means lower priority — subtitles with a level above `MinimumSubtitleLevel` are suppressed. See the guides for more on filtering.
    </Note>
  </Step>

  <Step title="Set up LocalAudioSubtitles in a LocalScript">
    Create a `LocalScript` named `LocalAudioSubtitlesSetup` inside `StarterPlayer > StarterPlayerScripts`. This script requires both `LocalAudioSubtitles` and `LocalAudio` from your `Packages` folder, then passes your subtitle data to `SetUp`.

    ```luau LocalAudioSubtitlesSetup.client.luau
    --!strict

    local ReplicatedStorage = game:GetService("ReplicatedStorage")
    local LocalAudioSubtitles = require(ReplicatedStorage:WaitForChild("Packages"):WaitForChild("LocalAudioSubtitles"))
    local LocalAudio = require(ReplicatedStorage:WaitForChild("Packages"):WaitForChild("LocalAudio"))
    local SubtitlesData = require(ReplicatedStorage:WaitForChild("Data"):WaitForChild("Subtitles"))

    LocalAudioSubtitles:SetUp(SubtitlesData :: any)
    LocalAudio:SetUp()
    ```

    `SetUp` connects LocalAudio's `ShowSubtitle` event to the Subtitle Engine renderer and registers your subtitle data with the event transform system.
  </Step>

  <Step title="Start LocalAudio on the server">
    Create a `Script` in `ServerScriptService` to initialize LocalAudio on the server side. Server-side setup only requires the module — no additional calls are needed here.

    ```luau LocalAudioSetup.server.luau
    --!strict

    require(game:GetService("ReplicatedStorage"):WaitForChild("Packages"):WaitForChild("LocalAudio"))
    ```

    Refer to the [LocalAudio documentation](https://github.com/IITPP-Roblox/LocalAudio) for how to define audio data and trigger playback from the server.
  </Step>

  <Step title="(Optional) Filter subtitles by level">
    If you want to suppress lower-priority subtitles — for example, hiding ambient sound captions by default — call `SetSubtitleLevel` before `SetUp`:

    ```luau
    -- Only show subtitles at level 1 (Dialog) or below.
    -- Subtitles with Level > 1 (e.g. ExtraAnnouncement = 2) are hidden.
    LocalAudioSubtitles:SetSubtitleLevel(1)
    LocalAudioSubtitles:SetUp(SubtitlesData :: any)
    LocalAudio:SetUp()
    ```

    Pass a number equal to or above your highest level value to show all subtitles. The default is `0`, which hides every leveled subtitle (since any level number greater than 0 will be suppressed).
  </Step>
</Steps>
